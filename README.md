# Smart-Assistant-Project
Smart Assistant for the Visually Impaired
This project is a comprehensive smart assistant designed to aid visually impaired individuals in interacting with their surroundings using Artificial Intelligence technologies.
Key Features

    Text Reading (OCR): Integrated PaddleOCR to extract and read printed text aloud.

    Object Detection: Utilized YOLO to detect surrounding objects and provide real-time audio feedback.

    Face Recognition: Implemented Dlib to identify familiar faces.

    Voice Guidance: Integrated an English text-to-speech engine to facilitate seamless interaction with the system.
    
  Technologies Used

    Language: Python

    Computer Vision: OpenCV, YOLO, Dlib

    OCR: PaddleOCR

    AI/ML: Deep Learning Models
